Lenguajes FrontEnd

HTML

Leguaje de marcado de hiptertexto que se utiliza para definir la estructura de las paginas web, utiliza etiquetas para establecer la funcion y jerarquia de los elementos de la misma. En los ultims años se hace mucho enfasis en la semantica, que permite a los motores de busqueda indizar los contenidos del sitio con mayor facilidad.

CSS

Las hojas de estilo en cascada se utilizan para definir la apariencia de los elementos que componen nuestra pagina web, utiliza selectores y reglas para aplicarse dentro de la pagina. Este lenguaje permite realizar modificaciones en la pagina a nivel visual y es ideal para proporcionar una grata experiencia de usuario, ademas de una interfaz que responda segun el tamaño de la pantalla que se visualice.

Javascript

Lenguaje de programacion orientado a objetos que se ejecuta directamente en el navegador, ya que no necesita ser compilado. Implementa ciertas caracteristicas que permiten la manipulacion de la pagina mediante eventos o acciones de usuario, gracias a los modelos DOM Y BOM que corresponden a Docment Object Model y Browser Object Model.
